INSTRUÇÕES PARA CRIAR REPOSITÓRIO NO GITHUB E FAZER PUSH
================================================================

1. CRIAR REPOSITÓRIO NO GITHUB:
   - Acesse: https://github.com/new
   - Nome do repositório: Diversidade-Genero-Ufes
   - Marque como: PRIVADO (Private)
   - NÃO inicialize com README, .gitignore ou licença
   - Clique em "Create repository"

2. DEPOIS DE CRIAR, EXECUTE ESTES COMANDOS NO TERMINAL:

   git remote add origin https://github.com/SEU-USUARIO/Diversidade-Genero-Ufes.git
   git push -u origin main

   (Substitua SEU-USUARIO pelo seu nome de usuário do GitHub)

OU, SE USAR SSH:

   git remote add origin git@github.com:SEU-USUARIO/Diversidade-Genero-Ufes.git
   git push -u origin main

================================================================
DICA: Após criar o repositório, o GitHub mostrará a URL exata
para você usar. Copie e cole no comando acima.
